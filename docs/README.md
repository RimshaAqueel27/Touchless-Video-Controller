## Adding Assets

When adding images or videos:
1. Use descriptive filenames (e.g., `gesture-detection-demo.gif`)
2. Optimize images for web (< 1MB preferred)
3. Update README.md references
4. Credit original sources if applicable

## Demo Recording Tips

1. Use tools like [ScreenToGif](https://www.screentogif.com/) or [LICEcap](https://www.cockos.com/licecap/)
2. Record at 10-15 FPS for smaller file size
